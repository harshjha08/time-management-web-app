# ⏰ Digital Clock Web App

A feature-rich and responsive Digital Clock application built using HTML, CSS, and Vanilla JavaScript. This project combines multiple time management tools including a real-time clock, alarm manager, stopwatch, countdown timer, and customizable themes in a modern glassmorphism-inspired user interface.

## ✨ Features

### 🕒 Real-Time Digital Clock

* Live clock updated every second
* Displays current day, date, and month
* Clean and minimal interface

### ⏰ Alarm Manager

* Create up to 3 alarms
* Custom alarm labels
* Duplicate alarm prevention
* Upcoming alarms section
* Alarm popup notification
* Alarm ringtone support
* Delete alarms individually
* Alarm persistence using Local Storage

### ⏱ Stopwatch

* Start, pause, and reset functionality
* Lap recording support
* Dynamic lap history display
* Accurate second-by-second tracking

### ⏳ Countdown Timer

* Set hours, minutes, and seconds
* Start, pause, and reset controls
* Countdown completion alert
* Responsive timer interface

### 🎨 Theme System

Choose between multiple built-in themes:

* Black
* Purple
* Blue
* Orange
* Green

Selected theme is automatically saved using Local Storage and restored on page reload.

### 📱 Responsive Design

* Mobile-friendly layout
* Dedicated mobile stylesheet
* Responsive navigation
* Optimized controls for smaller screens

## 🚀 How to Run

1. Clone the repository

```bash

```

2. Open the project folder

```bash
cd digital-clock
```

3. Launch `index.html` in your browser

No frameworks or installations required.
